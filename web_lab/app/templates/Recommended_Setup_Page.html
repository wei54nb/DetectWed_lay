系統配置建議

1. 硬體配備

CPU 需求

  • 最低需求：Intel Core i5-8400 或 AMD Ryzen 5 2600
  • 建議規格：Intel Core i7-9700K 或 AMD Ryzen 7 3700X 以上

GPU 需求

  • 最低需求：NVIDIA GTX 1660 6GB
  • 建議規格：NVIDIA RTX 3060 8GB 以上
  • 備註：需支援CUDA

記憶體與儲存空間

  • RAM最低需求：16GB
  • RAM建議規格：32GB
  • 儲存空間：至少 256GB SSD

其他硬體需求

  • 網路連線：穩定的寬頻網路，上傳速度至少 5Mbps
  • 相機規格：1080p/30fps 網路攝影機，建議使用 Logitech C920 或更高規格

2. 軟體需求

基礎軟體環境

  • 作業系統：Windows 10/11 64位元、Ubuntu 20.04 LTS 以上
  • Python版本：3.8 ~ 3.10
  • CUDA版本：11.7 或 12.1
  • PyTorch版本：2.0.0 以上（需與CUDA相容）

必要驅動程式

  • NVIDIA顯示卡驅動程式（支援CUDA）
  • 相機驅動程式
  • Visual C++ Redistributable（Windows環境）

3. 環境設置

虛擬環境設置

python -m venv venv
source venv/bin/activate # Linux/Mac
venv\Scripts\activate # Windows

建議套件安裝順序：

  1. CUDA Toolkit
  2. PyTorch
  3. 其他依賴套件

環境變數設定

  • CUDA_HOME 設定
  • PATH 變數更新
  • Python 環境變數

4. 其他建議

使用環境建議

  • 相機擺放：建議高度1.5-1.8公尺，與使用者保持2-3公尺距離
  • 光線需求：充足且均勻的照明，避免逆光
  • 運動空間：建議預留至少2x2公尺的活動空間
  • 背景建議：單色且無雜物的背景
